# Adii Vendor Invoice Intelligent System

Freight cost prediction and invoice risk flagging system built with FastAPI, LightGBM, and React.

## Features
- Freight cost prediction using ML
- Invoice risk flagging and scoring
- Customer management
- Real-time dashboard

## Tech Stack
- Backend: FastAPI, SQLAlchemy, PostgreSQL
- ML: LightGBM, MLflow, Feast
- Frontend: React, TypeScript
- Infrastructure: Docker, Kubernetes
